Welcome to the Meaningful Use Quality Measure Value Set Service

The Value Set Viewer is a desktop application style web application that allows users to browse value sets and their members. This web application connects to a value set service providing the 2014 Clinical Quality Measures prescribed for Meaningful Use of Electronic Health Records. The Viewer's functionality is designed to allow users to access value sets without explicit knowledge of the standards that created them and to make unambiguous choices of a given value set. User selected value sets can be downloaded in a variety of formats.

Viewing and/or downloading value sets requires a free Unified Medical Language System® Metathesaurus License, due to usage restrictions on some of the codes included in the value sets. Authentication based on acceptance of the license agreement is required. A dialog window will pop up before allowing access to a service. Licensing and authentication credentials are available at the NLM UTS License Center.

This Common Terminology Services 2 (CTS2) compliant value set service browser can connect to any [CTS2] REST enabled value set service provider. CTS2 is an OMG standard for the query, interchange and update of terminological resources. The standard is based on Restful Oriented Architecture (ROA).

This CTS2 compliant value set service provides a mirror of the eMeasures Meaningful Use value sets hosted by the U.S. National Library of Medicine (NLM). Users may alternatively access this value set data by accessing the NLM Value Set Authority Center (VSAC).
